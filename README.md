# FastoClick

Welcome to the FastoClick project!

## Social Media Integrations & APIs

We have successfully integrated Meta/Facebook APIs into the TWOM (The World of Matrix) ecosystem, providing seamless page posting, authentication, scheduling, and analytics.

You can find the complete list of Meta and Facebook APIs utilized in the project in the dedicated documentation:

* 📄 **[Meta/Facebook API Reference (META_APIS.md)](file:///c:/Utsav/ferolt/FastoClick/META_APIS.md)**

---

### Key Features Integrated for TWOM:
- **OAuth Login & User Authentication**
- **Short-Lived to Long-Lived Token Exchange**
- **Connected Facebook Pages Fetching & Token Generation**
- **Facebook Post Creation (Auto & Scheduled)**
- **Detailed Metrics & Post Analytics Tracking (Impressions, Reach, Likes, Comments, etc.)**
- **Media Uploads (Images & Videos)**

Feel free to browse [META_APIS.md](file:///c:/Utsav/ferolt/FastoClick/META_APIS.md) for full details on endpoints, fields, payloads, and examples.
